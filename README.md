# CTFd-Like

## Install & setup project

```
Step 1:
git clone https://github.com/AdrienIT/CTFd-like.git
cd CTFd-like
Edit the .env file to your preferences.

Step2
sudo bash setup.sh
OR
chmod +x setup.sh
sudo ./setup.sh

Step3
http://localhost/
OR
http://$IP/
```
